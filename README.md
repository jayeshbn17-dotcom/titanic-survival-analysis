# Titanic Survival Analysis

## Project Overview

This project analyzes the Titanic dataset to identify factors influencing passenger survival and build a machine learning model for survival prediction.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Tableau

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Logistic Regression Model
5. Model Evaluation
6. Tableau Dashboard Development

## Key Insights

- Female passengers had significantly higher survival rates.
- First-class passengers were more likely to survive.
- Higher ticket fares were associated with greater survival probability.
- Age showed moderate impact on survival.

## Model Performance

- Algorithm: Logistic Regression
- Accuracy: 83%
- Evaluation Metrics:
  - Confusion Matrix
  - Classification Report
  - ROC Curve

## Dashboard Features

- Gender vs Survival
- Fare vs Survival

- Interactive Filters
